## Лабораторная работа №1 "Nginx"

### Задачи
Выполнить развертывание и конфигурацию веб-сервера Nginx на локальной машине:
- Обеспечить работу веб-сервера по защищенному протоколу с 
использованием SSL сертификатов
- Реализовать принудительное перенаправление на HTTPS
- Обеспечить возможность одновременного обслуживания нескольких доменных имен на одном сервере
- Применить директиву `alias` для управления путями
